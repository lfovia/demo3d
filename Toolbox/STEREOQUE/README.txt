0) If you are using STEREOQUE code please cite this paper.
B. Appina, S. Khan, and S. S. Channappayya, "No-reference stereoscopic image quality assessment using natural scene statistics," Signal Processing Image Communication, vol. 43, pp. 1 - 14, 2016.

1) Download STEREOQUE code from http://www.iith.ac.in/~lfovia

2) Download the Steerable pyramid toolbox from http://www.cns.nyu.edu/pub/eero/matlabPyrTools.tar.gz 

3) Disparity map can be estimated using the code available at http://live.ece.utexas.edu/research/quality/MJ3DFR_release.zip 

4) Copy all the files present in the folder "STEREOQUE_Software" to the steerable pyramid toolbox folder

5) See example_stereque.m for testing. Note that the libsvm package is required for regression and can be downloaded from "https://www.csie.ntu.edu.tw/~cjlin/libsvm/"
